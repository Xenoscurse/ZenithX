Command Categories

Moderation

  • /ban - Ban users with reason
  • /kick - Remove users from server
  • /mute - Temporarily mute members
  • /warn - Issue warnings to users
  • /purge - Bulk delete messages

Tickets

  • /ticket setup - Create ticket panel
  • /ticket close - Close active ticket
  • /ticket add - Add user to ticket
  • /ticket remove - Remove user
  • /ticket transcript - Get logs

Utility

  • /userinfo - Get user details
  • /serverinfo - Server statistics
  • /avatar - Get user avatar
  • /ping - Check bot latency
  • /help - Command list & help

Frequently Asked Questions

How do I add Genos to my server?
Click the "Invite Genos" button above and select your server from the dropdown. You'll need "Manage Server" permissions to add the bot.
Is Genos free to use?
Yes! All features are completely free with no limitations. We believe in providing quality tools to the Discord community.
How do I get support?
Join our support server using the "Support Server" button above. Our community and staff will be happy to help you!
Can I suggest new features?
Absolutely! We welcome feature suggestions in our support server. The most requested features get priority in our development roadmap.

About Genos

Genos is a powerful multi-purpose Discord bot designed to enhance your server experience with premium features.


Key Features:

  • Advanced Moderation Tools
  • Customizable Ticket System
  • Welcome & Goodbye Messages
  • Server Analytics & Logging
  • Role Management & Auto-Roles
  • And much more!

All features are completely free to use with no limitations. Genos is constantly updated with new features and improvements based on community feedback.


Development Philosophy:

We believe in creating tools that empower server administrators and enhance community engagement. Our focus is on reliability, performance, and user experience.
